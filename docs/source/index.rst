requests-consul
===============

requests-consul is an extension for the
`requests <http://docs.python-requests.org/en/latest/>`_ library.

It's implementing a custom transport adapter which connects micro
services managed by `Consul <http://consul.io/>`_
(via the `service://` protocol).
